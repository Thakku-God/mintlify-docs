# File Limits

| File | Maximum Size |
| --- | --- |
| Combined YFT \+ YTD | **40 MB** |
| Single YTD | **5 MB** |

## Model Requirements

- Vehicle must be fully de-badged.
- Lore-friendly/custom branding only.
- Polygon count should be optimized Under **3,00,000 - 3,50,000**.
- Mesh should be triangulated.
- Liveries must not exceed **2048×2048**.

## Lighting

- Heavy lighting animations are prohibited.
- Emissive textures: Maximum **40%**.

## Collision Rules

- Bullets must pass through front and side windows.
- Ramp collisions are prohibited.
- Invisible collisions are prohibited.
- Collision must align with the mesh.

## Damage Requirements

- Wheels must burst normally.
- Vehicle should explode within approximately **3 AR magazines**.
- Petrol tank must leak after approximately **1 bullet**.

## Seat Position

Driver position must follow Rockstar standards.

## Meta Rules

- Rockstar hierarchy required.
- Standard layouts only.
- Maximum LOD Distance: **350.0**

## Handling

- Maximum speed: **300 KM/H**
- Balanced handling only.
