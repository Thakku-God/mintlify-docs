# File Limits

| Asset | Maximum Size |
| --- | --- |
| Total Resource Size | **80 MB** |

## Requirements

1. Interior must be optimized.
2. MLO/IPL should not cause FPS drops.
3. Unused props should be removed.
4. Proper portals and room definitions are recommended.
5. Collision must match the visible model.
6. No invisible walls or unfair hiding spots.
7. Lighting should be optimized.
8. Streaming distance should be reasonable.
9. Doors must function correctly.
10. All textures should be optimized.

## Restrictions

- No copyrighted or offensive content.
- No objects providing unfair PvP advantages.
- No excessively bright emissive materials.
- No scripts without Management approval.

## Submission

1. Submit files at least **24 hours** before restart.
2. Management may request optimization before approval.
3. Houses exceeding **80 MB** will not be accepted.
