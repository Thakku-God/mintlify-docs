## File Limits

| Asset | Maximum Size |
| --- | --- |
| Total Resource Size | **80 MB** |

### Requirements

1. Interior must be optimized.
2. MLO/IPL should not cause FPS drops.
3. Unused props should be removed.
4. Proper portals and room definitions are recommended.
5. Collision must match the visible model.
6. No invisible walls or unfair hiding spots.
7. Lighting should be optimized.
8. Streaming distance should be reasonable.
9. Doors must function correctly.
10. All textures should be optimized.

### Animation Rules

1. A maximum of **4 animations** are permitted per Gang/Club House.
2. Animated copy props may have a maximum of **2–4 copies**.
3. Animated copy props must use a maximum **LOD Distance of 60**.
4. Main animations (such as **Gang Logos, Gang References, or other primary animated objects**) may use a maximum **LOD Distance of 100**.

### LOD Distance Guidelines

| Object Type | Maximum LOD |
| --- | --: |
| Outer Shell / Base Structure | **800** |
| Major Exterior Objects (Towers, Vehicle Garage, Helipad, Entrance) | **500** |
| Exterior Props (Trees, Gates) | **100** |
| Small Exterior & Interior Props (Gardens, Flowers, <br />Grass, Street Lights, Interior Props, Photos, Decorations) | **60** |

### Restrictions

- No copyrighted or offensive content.
- No objects providing unfair PvP advantages.
- No excessively bright emissive materials.
- No scripts without Management approval.

### Submission

1. Submit files at least **24 hours** before restart.
2. Management may request optimization before approval.
3. Houses exceeding **80 MB** will not be accepted.
4. Houses that do not comply with the animation or LOD guidelines may be returned for optimization before approval.
